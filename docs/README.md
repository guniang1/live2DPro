# 北京语言大学本科生毕业论文LaTeX模板使用说明

## 📋 文件清单

本模板包含以下文件：

- `base.cls` - 文档类文件（核心格式定义）
- `main.tex` - 主文档示例
- `base.bib` - 参考文献数据库（可选）
- `base.bst` - 参考文献样式（可选）
- `logo1.png` - 北语校徽图标
- `logo2.png` - 北语校名书法字体图片
- `SimSun.ttf` - 宋体字体文件（需自行准备）
- `SimHei.ttf` - 黑体字体文件（需自行准备）
- `Times Regular.ttf` - Times New Roman字体文件（需自行准备）

## 🎯 格式要求说明

本模板严格按照docx模板中的格式要求编写，主要包括：

### 1. 页面设置

- 纸张：A4（210mm × 297mm）
- 页边距：上2.54cm，下2.54cm，左3.18cm，右3.18cm
- 页眉：小五号宋体，内容为"北京语言大学学士学位论文"
- 页脚：小五号宋体，格式为"第 X 页 共 Y 页"

### 2. 封面格式

- **外封**：包含校名图片、"学士学位论文"标题（黑体一号）、论文题目、学生信息表格
- **中文内封**：包含"北京语言大学学士学位论文"标题及年级、论文信息表格、完成日期
- **英文内封**：包含英文论文题目、学生姓名、指导教师、学院、学位类型等信息

### 3. 章节格式

- **章标题**：小二号黑体，加粗，居中，段前17磅，段后17磅，行间距2倍
- **一级标题（section）**：小三号黑体，加粗，左对齐，段前13磅，段后13磅
- **二级标题（subsection）**：四号黑体，不加粗，左对齐，段前13磅，段后13磅
- **正文**：小四号宋体，行间距固定值20磅，首行缩进2字符

### 4. 摘要格式

- 标题：小二号黑体，加粗，居中
- 正文：小四号宋体，行间距固定值20磅
- 关键词：小四号宋体，"关键词："三字用黑体

### 5. 目录格式

- 标题：小二号黑体，加粗，居中
- 章：黑体小四号
- 节：宋体小四号
- 缩进：一级标题缩进2em，二级标题缩进4em

### 6. 图表格式

- 图标题：五号字体，位于图下方，格式为"图 X.Y 标题内容"
- 表标题：五号字体，位于表上方，格式为"表 X.Y 标题内容"
- 公式编号：Times New Roman字体，右对齐，格式为"(X.Y)"

### 7. 参考文献格式

- 标题：五号黑体，加粗，左对齐
- 正文：五号字体，中文宋体，英文Times New Roman，单倍行距

### 8. 附录格式

- 标题：小二号黑体，加粗，居中，段前17磅，段后17磅

## 🚀 快速开始

### 第一步：准备字体文件

将以下字体文件放在模板目录下（与base.cls同一目录）：

- `SimSun.ttf` - 宋体
- `SimHei.ttf` - 黑体
- `Times Regular.ttf` - Times New Roman

> **字体获取方式**：
>
> - Windows系统：从 C:\Windows\Fonts 目录复制
> - macOS系统：从 /Library/Fonts 或 ~/Library/Fonts 目录复制
> - Linux系统：安装字体包或从其他系统复制

### 第二步：填写论文信息

在main.tex文件开头填写你的论文信息：

```latex
\title{你的论文题目}
\entitle{Your English Title}
\author{你的姓名}
\enauthor{Your Name}
\stunum{你的学号}
\grade{2022级}
\college{你的学院}
\encollege{Your College}
\major{你的专业}
\instructor{导师姓名}
\eninstructor{Supervisor Name}
\degreetitle{Bachelor of Arts}  % 或 Bachelor of Engineering 等
\date{二〇二六}{六}{}  % 年、月、日
```

### 第三步：编写论文内容

在main.tex的正文部分编写你的论文内容：

```latex
\chapter{引言}
你的引言内容...

\section{研究背景}
你的研究背景...

\subsection{相关工作}
相关工作内容...
```

### 第四步：编译

使用XeLaTeX编译（**必须使用XeLaTeX，不能使用PDFLaTeX**）：

```bash
xelatex main.tex
xelatex main.tex  # 编译两次以生成正确的目录和引用
```

或者在支持的LaTeX编辑器（如TeXstudio、Overleaf）中选择XeLaTeX编译器。

## 📝 使用技巧

### 1. 插入图片

```latex
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.8\textwidth]{图片文件名.png}
    \caption{图片标题}
    \label{fig:标签名}
\end{figure}
```

引用图片：`如图\ref{fig:标签名}所示`

### 2. 插入表格

```latex
\begin{table}[htbp]
    \centering
    \caption{表格标题}
    \label{tab:标签名}
    \begin{tabular}{|c|c|c|}
        \hline
        表头1 & 表头2 & 表头3 \\
        \hline
        数据1 & 数据2 & 数据3 \\
        \hline
    \end{tabular}
\end{table}
```

引用表格：`如表\ref{tab:标签名}所示`

### 3. 插入公式

```latex
\begin{equation}
    E = mc^2
    \label{eq:标签名}
\end{equation}
```

引用公式：`如式\ref{eq:标签名}所示`

### 4. 添加参考文献

直接在文档中使用 `\begin{thebibliography}`环境：

```latex
\begin{thebibliography}{99}
\bibitem{ref1}
作者. 文献标题[J]. 期刊名称, 年份, 卷(期): 页码.
\end{thebibliography}
```

引用文献：`文献\cite{ref1}指出...`

或使用BibTeX（需要准备base.bib文件）：

```latex
\bibliographystyle{base}  % 使用base.bst样式
\bibliography{base}        % 使用base.bib数据库
```

### 5. 添加附录

```latex
\appendix
\chapter{附录标题}
附录内容...
```

## ⚠️ 常见问题

### 1. 编译错误：找不到字体文件

**原因**：字体文件未放在正确位置或文件名不匹配
**解决**：确保SimSun.ttf、SimHei.ttf和Times Regular.ttf在与base.cls同一目录下

### 2. 中文显示为方框或乱码

**原因**：未使用XeLaTeX编译
**解决**：必须使用XeLaTeX编译器，不能使用PDFLaTeX

### 3. 目录页码不对

**原因**：只编译了一次
**解决**：需要编译两次：第一次生成辅助文件，第二次更新目录和引用

### 4. 图片无法显示

**原因**：图片路径不正确或格式不支持**解决**：

- 确保图片文件与main.tex在同一目录
- 推荐使用PNG或JPG格式
- 检查文件名大小写是否匹配

### 5. 参考文献格式不对

**原因**：参考文献格式需要手动调整
**解决**：参照GB/T 7714-2015标准或参考模板示例中的格式

### 6. 页眉页脚未显示

**原因**：封面页和声明页不显示页眉页脚是正常的
**解决**：从正文（目录后）开始才会显示页眉页脚

- **章节格式**：修改第191-209行的 `\ctexset`设置
- **目录格式**：修改第180-189行的目录设置
- **图表格式**：修改第212-223行的caption设置
